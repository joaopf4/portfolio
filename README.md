## Portfólio Pessoal

Neste repositório guardo os arquivos do meu porfólio de projetos e com um pouco sobre quem eu sou =)

Feito com ReactJS e estilizado com Styled-Components, nele tento mostrar minha forma de pensar a navegação de um usário e minha forma de organização do código por aqui.

Pra rodar o projeto localmente, basta clonar este repositório, 
abrir a pastar com o VSCode, e o no terminal rodar o comando: 
### `npm start`
